# isNullish

Determine whether the input value is `null` or `undefined`.

### Usage

```ts
import { isNullish } from 'rattail'

isNullish(null) // return true
isNullish(undefined) // return true
isNullish(123) // return false
```

### Arguments

| Arg     | Type  | Defaults |
| ------- | :---: | -------: |
| `value` | `any` |          |

### Return

|   Type    |
| :-------: |
| `boolean` |
