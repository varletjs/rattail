# isFunction

Determine whether the input value is a function

### Usage

```ts
import { isFunction } from 'rattail'

isFunction(() => {}) // return true
isFunction(123) // return false
```

### Arguments

| Arg     | Type  | Defaults |
| ------- | :---: | -------: |
| `value` | `any` |          |

### Return

|   Type    |
| :-------: |
| `boolean` |
