# isString

Determine whether the input value is a string

### Usage

```ts
import { isString } from 'rattail'

console.log(isString('rattail')) // return true
console.log(isString(123)) // return false
```

### Arguments

| Arg     | Type  | Defaults |
| ------- | :---: | -------: |
| `value` | `any` |          |

### Return

|   Type    |
| :-------: |
| `boolean` |
