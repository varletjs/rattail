# 快速开始

## 总览

`Rattail` 是一个 AI Agent 友好的前端工具链。

### 特性

- <span style="font-size: 0.82em">🧰</span> &nbsp; 140+ 工具函数，覆盖通用、字符串、数字、数组、对象、数学等场景
- <span style="font-size: 0.82em">🔧</span> &nbsp; CLI 工具链，支持发布、日志、Git Hooks、Commit Lint、API 生成
- <span style="font-size: 0.82em">⚙️</span> &nbsp; 开箱即用的 Oxlint + Oxfmt 配置预设，支持 Vite+ 集成
- <span style="font-size: 0.82em">🚀</span> &nbsp; 基于 axios 的渐进式请求工具，支持 Vue 组合式 API
- <span style="font-size: 0.82em">📏</span> &nbsp; 链式校验规则工厂，适配任意 UI 框架
- <span style="font-size: 0.82em">🏷️</span> &nbsp; 类型安全的枚举工具，内置 label、description，支持自定义字段扩展
- <span style="font-size: 0.82em">🌲</span> &nbsp; 可 Tree-shake，轻量，TypeScript 完整类型支持
- <span style="font-size: 0.82em">💪</span> &nbsp; 99%+ 单元测试覆盖率

### 安装

::: code-group

```shell [npm]
npm i rattail -S
```

```shell [yarn]
yarn add rattail
```

```shell [pnpm]
pnpm add rattail
```

:::

### Skills

我们提供了 [Agent Skills](https://github.com/varletjs/skills) 用于 Rattail 生态系统。通过安装这些 Skills，AI 编程助手可以更好地理解和使用 `Rattail` 及其相关工具，帮助你更高效地编写代码。
