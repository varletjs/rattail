# meanBy

通过对 `数组` 的每个元素应用一个函数以得出数值，计算数组的平均值。

### 使用

```ts
import { meanBy } from 'rattail'

meanBy([{ n: 4 }, { n: 6 }, { n: 8 }], ({ n }) => n)
// return 6
meanBy([10, 20, 30], (n) => n / 2)
// return 10
```

### 参数

| 参数  |         类型         | 默认值 |
| ----- | :------------------: | -----: |
| `arr` |        `T[]`         |        |
| `fn`  | `(val: T) => number` |        |

### 返回值

|   类型   |
| :------: |
| `number` |
