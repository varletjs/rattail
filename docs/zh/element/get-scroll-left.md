# getScrollLeft

获取元素或窗口的水平滚动位置。

### 用法

```ts
import { getScrollLeft } from 'rattail'

const scrollLeft = getScrollLeft(window)
```

### 参数

| 参数      | 类型                | 默认值 |
| --------- | ------------------- | ------ |
| `element` | `Element \| Window` |        |

### 返回值

| 类型     |
| -------- |
| `number` |
