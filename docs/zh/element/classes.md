# classes

根据给定条件生成类名列表，或直接返回类名。

### 用法

```ts
import { classes } from 'rattail'

const classList = classes('active', [true, 'visible', 'hidden'])
```

### 参数

| 参数      | 类型      | 默认值 |
| --------- | --------- | ------ |
| `classes` | `Classes` |        |

### 返回值

| 类型    |
| ------- |
| `any[]` |
