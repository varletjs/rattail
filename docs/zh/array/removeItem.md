# removeItem

从数组中移除第一次出现的指定元素，返回移除的元素。

### 使用

```ts
import { removeItem } from 'rattail'

const arr = [1, 2, 3]
removeItem(arr, 2) // arr 变为 [1, 3]
```

### 参数

| 参数   | 类型    | 默认值 |
| ------ | ------- | ------ |
| `arr`  | `Array` |        |
| `item` | `any`   |        |

### 返回值

| 类型  |
| ----- |
| `any` |
