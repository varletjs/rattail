# upperFirst

将`字符串`的`首字母`大写，其余部分保持不变。

### 使用

```ts
import { capitalizeFirstLetter } from 'rattail'

capitalizeFirstLetter('hello world') // return 'Hello world'
capitalizeFirstLetter('rattail') // return 'Rattail'
```

### 参数列表

| 参数    |   类型   | 默认值 |
| ------- | :------: | -----: |
| `value` | `string` |        |

### 返回值

|   类型   |
| :------: |
| `string` |
