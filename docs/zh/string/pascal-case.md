# pascalCase

将字符串转换为 `PascalCase`。

### 使用

```ts
import { pascalCase } from 'rattail'

pascalCase('hello-world') // return 'HelloWorld'
pascalCase('fooBar') // return 'FooBar'
```

### 参数列表

| 参数    |   类型   | 默认值 |
| ------- | :------: | -----: |
| `value` | `string` |        |

### 返回值

|   类型   |
| :------: |
| `string` |
