# lowerFirst

将`字符串`的`首字母`小写，其余部分保持不变。

### 使用

```ts
import { lowerFirst } from 'rattail'

upperFirst('Hello') // return 'hello'
upperFirst('Rattail') // return 'rattail'
```

### 参数列表

| 参数    |   类型   | 默认值 |
| ------- | :------: | -----: |
| `value` | `string` |        |

### 返回值

|   类型   |
| :------: |
| `string` |
