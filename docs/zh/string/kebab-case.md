# kebabCase

将字符串转换为 `kebab-case` 格式

### 使用

```ts
import { kebabCase } from 'rattail'

kebabCase('HelloWorld') // return 'hello-world'
kebabCase('fooBar') // return 'foo-bar'
```

### 参数列表

| 参数    |   类型   | 默认值 |
| ------- | :------: | -----: |
| `value` | `string` |        |

### 返回值

|   类型   |
| :------: |
| `string` |
