# copyText

Copies text to the clipboard.

### Usage

```js
import { copyText } from 'rattail'

copyText('Hello, clipboard!') // The string will be written to the clipboard
```

### Arguments

| Arg    | Type     | Defaults |
| ------ | -------- | -------- |
| `text` | `string` |          |

### Return

| Type |
| ---- |
| `-`  |
