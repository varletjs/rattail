# lowerFirst

Lowercase the `first letter` of the `string` and keep the rest unchanged.

### Usage

```ts
import { upperFirst } from 'rattail'

upperFirst('Hello') // return 'hello'
upperFirst('Rattail') // return 'rattail'
```

### Arguments

| Arg     |   Type   | Defaults |
| ------- | :------: | -------: |
| `value` | `string` |          |

### Return

|   Type   |
| :------: |
| `string` |
