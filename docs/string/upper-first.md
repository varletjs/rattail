# upperFirst

Capitalize the `first letter` of a `string`, leaving the rest of the string unchanged.

### Usage

```ts
import { upperFirst } from 'rattail'

upperFirst('hello world') // return 'Hello world'
upperFirst('rattail') // return 'Rattail'
```

### Arguments

| Arg     |   Type   | Defaults |
| ------- | :------: | -------: |
| `value` | `string` |          |

### Return

|   Type   |
| :------: |
| `string` |
