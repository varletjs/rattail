# vite-plus-presets

Opinionated oxlint + oxfmt configuration preset for vite-plus projects. Integrated with [@configurajs/vite-plus](https://github.com/configurajs/vite-plus).

### Usage

```ts
import { lint, fmt } from 'rattail/vite-plus'
```
