# sum

Calculates the sum of values in an array of numbers.

### Usage

```ts
import { sum } from 'rattail'

sum([1, 2, 3]) // returns 6
```

### Arguments

| Arg   | Type       | Defaults |
| ----- | ---------- | -------- |
| `arr` | `number[]` |          |

### Return

| Type     |
| -------- |
| `number` |
