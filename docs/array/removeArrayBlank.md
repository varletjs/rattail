# removeArrayBlank

Removes `null` or `undefined` values from an array.

### Usage

```ts
import { removeArrayBlank } from 'rattail'

removeArrayBlank([1, null, 2, undefined, 3]) // returns [1, 2, 3]
```

### Arguments

| Arg | Type  | Defaults |
| --- | ----- | -------- | --- |
|     | `arr` | `Array`  |     |

### Return

| Type    |
| ------- |
| `Array` |
