# at

Retrieves the element at a specified index in an array, supporting negative indices.

### Usage

```ts
import { at } from 'rattail'

at([1, 2, 3], -1) // returns 3
```

### Arguments

| Arg     | Type     | Defaults |
| ------- | -------- | -------- |
| `arr`   | `Array`  |          |
| `index` | `number` |          |

### Return

| Type  |
| ----- |
| `any` |
