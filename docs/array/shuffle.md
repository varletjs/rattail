# shuffle

Randomly shuffles elements within an `array`.

### Usage

```ts
import { shuffle } from 'rattail'

shuffle([1, 2, 3]) // return shuffled array
```

### Arguments

| Arg   | Type    | Defaults |
| ----- | ------- | -------- |
| `arr` | `Array` |          |

### Return

| Type    |
| ------- |
| `Array` |
