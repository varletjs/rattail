# shuffle

Randomly shuffles elements within an array.

### Usage

```ts
import { shuffle } from 'rattail'

shuffle([1, 2, 3]) // returns [2, 1, 3] (output may vary)
```

### Arguments

| Arg   | Type    | Defaults |
| ----- | ------- | -------- |
| `arr` | `Array` |          |

### Return

| Type    |
| ------- |
| `Array` |
