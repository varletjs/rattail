# inViewport

Determines if an element is visible within the viewport.

### Usage

```ts
import { inViewport } from 'rattail'

const isVisible = inViewport(document.querySelector('div'))
```

### Arguments

| Arg       | Type          | Defaults |
| --------- | ------------- | -------- |
| `element` | `HTMLElement` |          |

### Return

| Type      |
| --------- |
| `boolean` |
