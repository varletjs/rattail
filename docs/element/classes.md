# classes

Generates a list of class names based on a given condition or directly returns class names.

### Usage

```ts
import { classes } from 'rattail'

const classList = classes('active', [true, 'visible', 'hidden'])
```

### Arguments

| Arg       | Type      | Defaults |
| --------- | --------- | -------- |
| `classes` | `Classes` |          |

### Return

| Type    |
| ------- |
| `any[]` |
