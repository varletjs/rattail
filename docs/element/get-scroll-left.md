# getScrollLeft

Gets the horizontal scroll position of an element or window.

### Usage

```ts
import { getScrollLeft } from 'rattail'

const scrollLeft = getScrollLeft(window)
```

### Arguments

| Arg       | Type                | Defaults |
| --------- | ------------------- | -------- |
| `element` | `Element \| Window` |          |

### Return

| Type     |
| -------- |
| `number` |
