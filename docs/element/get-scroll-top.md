# getScrollTop

Gets the vertical scroll position of an element or window.

### Usage

```ts
import { getScrollTop } from 'rattail'

const scrollTop = getScrollTop(window)
```

### Arguments

| Arg       | Type                | Defaults |
| --------- | ------------------- | -------- |
| `element` | `Element \| Window` |          |

### Return

| Type     |
| -------- |
| `number` |
